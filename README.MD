

A backend API for managing vehicles, trips, and users with role-based access control.  
Built using **Node.js**, **Express.js**, and **Supabase (PostgreSQL)**.

---


Design and build a **Fleet Management Backend API** that supports:
- Multiple user roles
- Vehicle management
- Trip management
- Secure authentication & authorization

---


- **Node.js**
- **Express.js**
- **Supabase (PostgreSQL)**
- **JWT Authentication**
- **bcrypt**
- **dotenv**

---



The system supports **three roles**:

- **Admin**
- **Manager**
- **Driver**

### Role Permissions:
| Role     | Permissions |
|--------|-------------|
| Admin   | Manage users, vehicles, and trips |
| Manager | Manage vehicles and trips |
| Driver  | View assigned trips |
